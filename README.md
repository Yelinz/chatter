# chatter

ChatGPT voice interface with some translation features.

Built as a learning project with flutter.
The idea was to have an android, ios and web version with just one repository.

To use this you have to provide your own API keys for the different services in the .env file.
and then build the apk to install on your phone.
```
flutter build apk
```
